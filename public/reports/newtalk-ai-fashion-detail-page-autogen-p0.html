요약

뉴톡 AI 패션 시스템의 마지막 제작 단계인 상세페이지 자동생성을 전담하는 기획서입니다. 사용자 흐름 ①~⑥ 단계에서 확정된 승인 모델컷·감성컷·상품 정보·코디 세트를 입력받아, V1 뉴톡 상품 상세페이지와 호환되는 HTML을 자동 조립합니다.

핵심 원칙: 상세페이지는 "이미지 나열"이 아니라 판매 스토리 구조입니다. 히어로 → 핵심 카피 → 착용 핏 → 소재 디테일 → 코디 제안 → 사이즈 가이드 → CTA 순서로, 고객이 스크롤 한 번에 구매 결정에 필요한 정보를 모두 얻도록 설계합니다.

자동생성 범위

  • 승인 이미지 자동 배치 (히어로, 착용, 디테일, 코디)
  • AI 카피라이팅 (상품명 기반 판매 문구 생성)
  • 블록 기반 템플릿 자동 조립
  • 모바일/PC 반응형 HTML 출력
  • V1 상품등록 시스템 호환 포맷

수동 편집 범위

  • 블록 순서 변경, 추가, 삭제
  • 카피 문구 수정·교체
  • 이미지 교체·크롭·순서 변경
  • CTA 버튼 위치·문구 조정
  • 최종 판매 반영 승인

기대 효과

  • 상세페이지 제작 시간: 2~4시간 → 15~30분
  • 디자이너 의존도 제거 (운영자 단독 완성)
  • 일관된 브랜드 톤·레이아웃 유지
  • A/B 테스트로 전환율 최적화 가능
  • 도매몰(신상마켓 등) 동시 등록

V1 상세페이지 호환 기준

뉴톡 V1의 상품 상세페이지는 goods_content 필드에 HTML을 저장하고, 이 HTML이 상품 상세 화면에 그대로 렌더링되는 구조입니다. 자동생성 결과물은 이 구조를 100% 호환해야 합니다.

블록 기반 상세페이지 구조

상세페이지를 13개 표준 블록으로 구성합니다. 각 블록은 독립적으로 추가·제거·순서 변경이 가능하며, 자동생성 시 상품 속성에 따라 필요한 블록만 조립됩니다.

#블록명필수입력 소스자동생성 내용

AI 카피라이팅

각 블록에 필요한 판매 문구를 AI가 자동 생성합니다. 모델 페르소나의 말투를 반영하되, 과장 광고·허위 효능·가격 오인을 유발하는 문구는 자동 차단합니다.

블록카피 유형길이입력 데이터금지 표현

카피 품질 자동 검증

✅ 통과 조건

  • 금지 표현 패턴 매칭 0건
  • 소재 비율이 상품 DB와 일치
  • 가격 언급이 실제 판매가와 일치
  • 글자 수 제한 준수

❌ 자동 재생성 (최대 3회)

  • 금지 표현 1개 이상 감지
  • confidence < 0.7
  • 소재/가격 불일치
  • 글자 수 초과
