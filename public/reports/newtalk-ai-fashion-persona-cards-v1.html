연령/정체성 모든 모델은 한국 여성 20~30대 성인으로 고정. 미성년자처럼 보이는 외형과 표현 금지. 모델 카드, 프롬프트, 이미지 검수 체크리스트에 age_band 저장.
외형 생성 실존 인물, 유명인, 기존 인플루언서와 유사한 reference 금지. 자체 생성 seed와 prompt 저장. 유사성 검수 통과 전 공개 금지. 생성 asset별 approved_at 저장.
상품 매칭 모델별 50개씩 1차 총 300개 후보 선별. 가격, 카테고리, 핏 리스크, 이미지 품질 기준으로 점수화. persona_product_matches에 match_score, reason, status 저장.
콘텐츠 승인 AI 생성 이미지, 상품 설명, 광고 문구는 게시 전 사람 승인. 핏/소재/사이즈 허위 주장 금지. 콘텐츠별 reviewer, approved_at, disclosure_text 저장.
표시 문구 AI 모델 표시와 광고/상품추천 표시를 모두 적용. 플랫폼별 라벨 기능이 있으면 병행. ai_disclosure_logs에 채널, 문구, 적용 위치 기록.
성과 측정 조회수보다 상품 클릭, 저장, 장바구니, 구매 전환, 반품률을 우선. 모델별 주간 리포트와 상품 교체 의사결정 로그 생성.

1차 공개 순서

1주차는 윤나리와 강루아 2명만 공개합니다. 윤나리는 구매 전환, 강루아는 노출 확산을 검증합니다. 2주차에 한시아와 박하루를 추가하고, 3주차에 서준희와 최미나를 추가해 객단가와 반품률 지표를 확인합니다.
